## 1.5 Java语言的特点

* **完全面向对象**：封装、继承、多态，面向对象编程，达到更好的高内聚，低耦合

* **支持分布式**：唤醒和分发任务

* **健壮型**：强类型机制、异常处理、垃圾的自动收集等保证。对指针的丢弃是明智的选择
  > 学习了解垃圾回收的算法

* **安全**：Java提供了安全机制防止恶意代码攻击，如安全防范机制（Classloader）

* **跨平台性**：Java应用程序不用关心不同的操作系统，但是不同的系统可以安装不同的虚拟机，这样就可以实现Java的字节码文件一次编写，处处运行的可跨平台性

## 1.6 Java跨平台的原理

   不同的操作系统，可以安装不同的JVM，编写的Java程序是运行在JVM上，所以开发者不用在意具体平台，屏蔽了操作系统

* JDK，Java Development Kits,Java开发工具包

* JRE:Java Runtime Environment,Java运行环境

* JVM:Java Virtual Machine

JDK=JRE+开发工具
JRE=JVM+核心类库
开发工具(bin)：javac.exe、java.exe、javadoc.exe
核心类库：lang and util包等工具类

* JVM不是只有一种，它有很多品牌，Oracle（sun）公司用到的是HotSpot Client and Server VM

* JVM可以支持很多语言，不止Java，只要能将其编译成字节码文件
